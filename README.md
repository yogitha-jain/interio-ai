This is the AI part of our interior design project.  
It integrates:
- Stable Diffusion (for design generation)
- YOLOv8 (for object placement detection)
- OpenCV (for furniture detection and dimension estimation)

## Main Files
| File | Description |
|------|--------------|
| app.py | Main Flask API for AI model interface |
| cost_estimator.py | Estimates design/furniture cost |
| dimension_estimator.py | Measures furniture or room size |
| suggestion_engine.py | Suggests layout or design improvements |
| requirements.txt | Python dependencies |
